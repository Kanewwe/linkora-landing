Common Questions

常見問題解答

Vendor 一定要簽約嗎？ +
是的，Vendor 需要簽訂合作協議，確認寄信上限、抽成比例、付款方式以及服務條款。
可以用自己的 API Key 嗎？ +
可以！我們支援自備 Apollo.io, Hunter.io API Key 以及 SMTP 通道。這樣您的數據掌握在自己手中。
名單資料會被其他用戶看到嗎？ +
不會。每個用戶都有獨立的「私域池」，全域池的資料皆經過脫敏處理，保護隱私。
支援哪些語言？ +
目前支援繁體/簡體中文、英文。AI 劇本功能可自動產出多語言開發信。
數據合規嗎？ +
我們遵守 GDPR、CCPA 與台灣個資法，僅採集公開資訊，不涉及私密數據。
